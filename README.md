## これは何？

Skkeleton の状態を入力中に判別しやすいよう、カーソルの側にモードを表示します。AquaSKK にあるようなやつです。

NOTE: Skkeleton は Neovim/Vim 両対応ですが、このプラグインは Neovim 専用です。

## 使い方

```lua
require'skkeleton_indicator'.setup{}
```

オプションなどは doc を見てください。
